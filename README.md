# Navigation Menu Hover Animation

Projeto desenvolvido com HTML e CSS que demonstra diferentes efeitos de animação em menus de navegação utilizando apenas CSS. Ao passar o cursor sobre os itens do menu, animações suaves são executadas para destacar a navegação.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3

## ✨ Funcionalidades

- Menu de navegação horizontal
- Efeitos Hover animados
- Transições suaves
- Interface moderna
- Estrutura simples e reutilizável
- Sem utilização de JavaScript

## 📂 Estrutura do Projeto

```bash
Navigation-Menu-Hover-Animation/
│
├── index.html
├── style.css
└── README.md
```

## 🎯 Como Executar

1. Clone ou baixe o projeto.
2. Abra a pasta do projeto.
3. Execute o arquivo `index.html` no navegador.

## 📚 Conceitos Praticados

- HTML Semântico
- Listas (`ul` e `li`)
- Pseudo-classes (`:hover`)
- CSS Transitions
- Animações CSS
- Posicionamento de elementos
- Design de Navegação

## 💡 Objetivo

Praticar animações de interface e efeitos hover em menus de navegação, melhorando conhecimentos em CSS e experiência do usuário (UX).

## 🎨 Recursos Visuais

- Destaque visual ao passar o mouse
- Efeito de transição suave
- Navegação limpa e minimalista
- Layout moderno
